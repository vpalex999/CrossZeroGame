Правила игры в Крестики-Нолики.

Логическая игра между двумя противниками на квадратном поле 3х3 клетки или
большего размера.

- Один из игроков играет "крестиками", второй - "ноликами".

- Игроки по очереди ставят на свободные клетки поля знаки (один всегда крестики, другой всегда нолики).

- Первый, выстроивший в ряд 3 своих фигуры по вертикали, горизонтали или диагонали, выигрывает.

- Первый ход делает игрок, ставящий крестики.

# ---------------------- Преценденты (Варианты использования) ------------------------------------

UC.Запуск игры.
    1. И.запускает программу.
    2. С.выполняет внутреннюю инициализацию.
    3. С.инициализирует игровое поле.
        - создается GUI форма, которая содержит:
            - игровое поле с клетками.
            - два поля со статусом активного игрока, указывает, чей ход активный (Игрок х всегда ходит первый).
            - кнопку СТОП
    4. С.передаёт ход игроку Х.


UC.Начать снова.
    1. С.выполняет начальную инициализацию.
    2. С.Передает ход игроку Х.


UC.Ход игрока.
Цель: Занять клетку поля
    1. И. занимает свободную клетку кликая мышкой по ней.
    2. С. закрашивает клетку в символ игрока (Х или 0).
    3. С. проверяет есть ли выигрыш.
    4. С. передаёт ход второму игроку.

    Расширения:
    1.1. игрок пытается занять уже занятую клетку.
        С. Не меняет статус уже занятой клетки.

    3.1. Выигрыш есть.
        - С. закрашивает выигрышный ряд
        - С. выводит диалоговое окно с сообщением "Победа игрока .." и выбором "Начать снова" и "Выйти из игры".

    3.2. Выигрыша нет и все клетки заняты.
        - С. выводит диалоговое окно с сообщением "Ничья" и выбором "Начать снова" и "Выйти из игры".


UC.Прерывание игры.
    1. И.нажимает кнопку СТОП.
    2. С.выводит диалоговое окно с сообщением "Продолжение игры" и выбором "Начать снова" и "Выйти из игры".
    3. И.выбирает "Выйти из игры"
    4. С.завершает игру и работу программы.

    Расширения:
        2.1. П.закрывает диалоговое окно без выбора.
            - С.ожидает от И. выбора клетки.
        3.1. И.выбирает "Начать снова".
            - С. переходит в UC.Начать снова.
