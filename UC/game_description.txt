Правила игры в Крестики-НоликП.

Логическая игра между двумя противниками на квадратном поле 3х3 клетки или
большего размера.

- Один из игроков играет "крестиками", второй - "ноликами".

- Игроки по очереди ставят на свободные клетки поля знаки (один всегда крестики, другой всегда нолики).

- Первый, выстроивший в ряд 3 своих фигуры по вертикали, горизонтали или диагонали, выигрывает.

- Первый ход делает игрок, ставящий крестикП.

# ---------------------- Преценденты (Варианты использования) ------------------------------------

UC.Запуск игры.
    1. П.запускает программу.
    2. С.выполняет "Первоначальную инициализацию".
    3. Передаёт ход игрокуХ.

UC.Начать снова.
    Предусловия:
        П.нажал кнопку "СТОП".

    1. С.выполняет "Первоначальную инициализацию".

UC.Первоначальная инициализация.

    1. С.инициализирует "Статус активного игрока".
    2. С.инициализирует игровое поле.
    3. С.инициализирует игровое поле GUI.
        - создается GUI форма, которая содержит:
            - игровое поле с клетками.
            - два поля со статусом активного игрока, указывает, чей ход активный (Игрок х всегда ходит первый).
            - кнопку СТОП.


UC.Ход игрока.
    1. С.проверяет, возможность хода.
    2. П.занимает клетку на игровом поле.
    3. С.выполняет обработку хода и отмечает выбранную клетку за текущим игроком (Х или 0).
    4. С.проверяет есть ли выигрыш.
    5. С.меняет статус игрока на текущего.

    Расширения:
    
    1.1 Все клетки заняты, нет возможности сделать ход.
        - С. выводит диалоговое окно с сообщением "Ничья"
        -С.инициализирует UC.Выбор продолжения игры.

    2.1. П. прерывает игру.
        -С. выполняет UC.Прерывание игры.
    
    2.2. П.пытается занять уже занятую клетку.
        - С. Не меняет статус уже занятой клетки, переход хода не выполняется.

    4.1. Выигрыш есть.
        - С. закрашивает выигрышный ряд
        - С. выводит диалоговое окно с сообщением "Победа игрока ..".
        - С. инициирует UC.Выбор продолжения игры.



UC.Выбор продолжения игры.
    1. С.выводит диалоговое окно с сообщением "Продолжение игры" и выбором "Начать снова" и "Выйти из игры".\
    2. П.выбирает "Выйти из игры".
    3. С.завершает игру и работу программы.

    Расширения:
        2.1. П.выбирает "Начать снова"..
            - С. инициирует UC.Первоначальная инициализация.
        2.1. П.закрывает диалоговое окно крестиком.
            - С.отображает возвращается в последнее состояние игры.


UC.Прерывание игры.
    1. П.нажимает кнопку СТОП.
    2. С. инициирует UC.Выбор продолжения игры.
